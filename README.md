🎬 Java API Movie Search

Aplicação em Java que consome uma API externa de filmes para realizar buscas por título, converter respostas JSON em objetos Java utilizando Gson e armazenar histórico de pesquisas.

🚀 Tecnologias Utilizadas

Java 17+

HttpClient (java.net.http)

Gson

OMDb API

📌 Funcionalidades

✔ Buscar filme por nome
✔ Consumir API externa via HTTP
✔ Converter JSON em objeto Java
✔ Armazenar histórico de buscas
✔ Exibir lista de filmes pesquisados

🧠 Conceitos Aplicados

Programação Orientada a Objetos (POO)

Consumo de API REST

Desserialização de JSON

Uso de bibliotecas externas

Manipulação de listas

Estrutura de menu interativo no console

📂 Estrutura do Projeto
src/

▶ Como Executar

Clone o repositório:

git clone https://github.com/seuusuario/java-api-movie-search.git

Abra o projeto na sua IDE (IntelliJ, Eclipse ou VS Code)

Execute a classe Main

💡 Exemplo de Uso
1 - Buscar filme
2 - Listar histórico
0 - Sair

Digite o nome do filme e o sistema retornará as informações encontradas.
